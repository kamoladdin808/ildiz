# Меню Ildiz (myresto.online)

**API:** `https://ildiz-milliytaomlari.myresto.online/api/v1/menu`

**Категорий в экспорте:** 28 (только публичные). **Позиций:** 275 (без `isHidden`).
**Цена:** размер по умолчанию, UZS.

На сайте порядок вкладок и перенос позиций между разделами (в т.ч. «Соленья», «Паста») задаётся в `scripts/menu-print-layout.json` и применяется при сборке.

Чтобы обновить данные меню на сайте: после правок `MENU_MYRESTO_EXPORT.md` и/или `scripts/menu-print-layout.json` выполните `node scripts/build-menu-data.mjs` (генерируется `menu-data.js` для `menu.html`). Фото: совпадение по имени файла в `ildiz_menu/`, при необходимости `menu-photo-overrides.json`. Без снимка подставляется `ildiz_menu/no-photo.svg`. Старое поведение «раздать остатки файлов по порядку»: переменная окружения `MENU_PHOTO_POOL=1`.
